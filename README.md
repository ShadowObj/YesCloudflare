# YesCloudflare

查询Cloudflare反代节点小工具

程序原型由 [Joey Huang](https://t.me/Joeyblog/) 开发

Telegram反馈群: https://t.me/+ft-zI76oovgwNmRh/

```
Usage of ShadowObj/yescloudflare:
  
  -A    自动获取下一页内容 (默认需要确认)
  -asn int
        指定ASN
  -auto
        自动获取下一页内容 (默认需要确认)
  -file string
        指定输出文件 (默认ip.txt) (default "ip.txt")
  -key string
        指定API密钥
  -norepeat
        自动去除重复IP (默认不去除)
  -port int
        指定端口 (默认全部)
  -region string
        指定地区
  --help 
        获取帮助
```
